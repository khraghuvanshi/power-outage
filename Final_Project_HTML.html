Power Outage Analysis

Exploring Factors Affecting Power Outage Duration

Introduction

This project investigates the duration of power outages in different climate regions. It aims to identify key factors that impact outage length based on personal experiences and available data.

Hypothesis Testing

Null Hypothesis: There is no relationship between climate category and power outage duration.
Alternative Hypothesis: Extreme temperature conditions have on average longer outage durations.
Test Statistic: TVD (Total Variation Distance)

Baseline Model

The baseline model uses a simple linear regression approach with features like climate category and postal codes. Plans for improvement include adding more climate-specific variables and tuning hyperparameters.
